# RPG Tile Map Generator

A simple tile map generator for 2D sprite RPG games built with Python and Pygame.

## Features

- Place different tiles on a grid to create game maps
- Save maps to file for use in your game
- Load existing maps for editing
- Export maps as PNG images
- Simple and intuitive UI

## Installation

1. Clone this repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python main.py
   ```

## Usage

- Left-click to place selected tile
- Right-click to erase tile
- Scroll through available tiles using the sidebar
- Use the toolbar to save/load maps and export images
